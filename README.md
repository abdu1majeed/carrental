# CarRental  
A full-featured car rental platform built using Django.  
The system allows users to browse available cars, create accounts, book rentals, and manage their profiles.  
The project demonstrates clean architecture, responsive UI, authentication, authorization, and best practices.

---

## Project Description
CarRental is a modern web application that simplifies the process of browsing, booking, and managing rental cars.  
The project showcases your ability to build a complete Django app using templates, models, static & media files, authentication, authorization.

The application includes intuitive UI/UX optimized for all screen sizes, supporting secure user login, rental management, and admin controls.

---

## Features
- User Authentication (Register, Login, Logout)
- User Profiles
- Browse Cars with pagination
- Car Details Page
- Add/Edit/Delete Car listings (Admin only)
- Permission-based access (normal user vs admin)
- Media upload support for car images 
- Fully responsive UI using Bootstrap
- Template inheritance for clean structure
- Organized apps: main, accounts, rentals, bookings.
- Real dynamic URLs for pages & details
- Environment variables managed with python-dotenv
- Git & GitHub version control (daily commits)

---

## User Stories

### Regular User
- I can create an account.
- I can log in and log out.
- I can browse available cars.
- I can view car details.
- I can book a car.
- I can edit my profile information.

### Admin
- I can access a full dashboard.
- I can add new cars.
- I can update existing car info.
- I can delete cars.
- I can manage users.
- I can manage permissions.
- I can view and manage all rental bookings.
